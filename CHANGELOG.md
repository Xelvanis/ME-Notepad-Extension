# Changelog

## 1.0.0

- Initial standalone release of ME Notes.
- Adds local browser-saved notes for Marinara chats.
- Supports global, character, chat, branch, and all-branches note tabs.
- Includes Markdown preview, checklist toggles, formatting helpers, tab rename/reorder, layout reset, and backup import/export.
- Autosaves note edits silently so the notepad does not nag you every time you type.
